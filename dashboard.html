Slack Connection Guide

1. Configuration

Ensure config.env contains your SLACK_BOT_TOKEN (xoxb-) and SLACK_APP_TOKEN (xapp-).

2. Usage

Mention @Rover in any channel the bot is invited to.

  • @Rover ping! - Quick connectivity test.
  • @Rover [instruction] - Sends a command to the Rover agent.

3. 24-Hour Cycle

Note: Agents like Git and Diary only run once every 24 hours. Rover manages this automatically. Slack remains real-time.
